# za-bota-test
Тестовое задание для компании ZA|BOTA.

## Техническое задание:

1. Бот должен отвечать на /start и /help команды.
2. При отправке любого текстового сообщения бот должен обращаться к API сервиса ChatGPT и генерировать ответ на основе этого запроса.
3. Бот должен сохранять историю диалогов и использовать контекст предыдущих сообщения для формирования более качественных ответов.
4. Контекст диалога должен сбрасываться при вводе команды start, а также по нажатию кнопки Новый запрос.

## Развертывание в Docker:

1. Создайте .env файл на основе .env.example
2. Из корня проекта запустите 
    ```bash 
   docker compose up -d --build db
    ```

## Ручное развертывание:
1. Создайте переменные среды на основе .env файла
2. Из корня проекта выполните команду для установки пакетов
    ```bash 
   pip install -e .
    ```
3. Для запуска бота выполните команду
    ```bash 
   python -m bot
    ```
